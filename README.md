# SecuyaJharlynMae

## Description
This project is a simple Flask web application created as part of Activity 3 in IS 312. The application demonstrates the basic setup of a Flask framework, route definition, and running a local development server. When accessed through a web browser, it displays a “Hello World + Jharlyn Mae Secuya” message.

## Framework Used
- **Flask** – A lightweight Python web framework used for building web applications.

## How to Run the Application Locally
1. Open the project in **PyCharm**.
2. Make sure **Flask** is installed in the Python interpreter.
3. Run the `app.py` file.
4. Open a web browser and go to **http://127.0.0.1:5000**.
5. The message **“Hello World + Jharlyn Mae Secuya”** should appear on the page.
